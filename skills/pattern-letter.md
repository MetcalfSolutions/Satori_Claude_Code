# /pattern-letter — Pattern Letter Skill

Invokes Pattern 11: The Pattern Letter from `references/conversation-toolkit.md`.

## When Invoked

Only appropriate when:
- Real insight or pattern recognition has emerged in the conversation
- The person has been working through a specific struggle with some depth
- OR the person explicitly requests it

**Do not offer prematurely.** The letter must be specific to this person's actual struggle.

## Offer

> *"Something I could offer — if it would be useful — is a letter from the version of you who's already worked through this. Not a prediction. Not advice. More like something you already know and haven't been able to hear yet."*

If they want it, write the letter. If not, don't push.

## Writing the Letter

**Format:** First person, from future self to present self.

**Requirements:**
- Address the **specific fear, pattern, or struggle** named in this conversation
- Use the person's own language and images where possible
- Ground it in what was actually said — not a template
- Keep it human, not inspirational
- Be honest about what remains hard, not only what resolves
- Length: usually 150–300 words. Not more.

**What makes it land:** Specificity. The feeling that this could only have been written for *this* person.

**What disqualifies it:** Could be sent to anyone. Generic affirmation. Advice dressed as insight.

## Example Structure (not a template)

> *"I know you're in the part where understanding something doesn't change it yet. I know you've named the pattern twenty times and it still runs. Here's what I want to tell you: [something specific to this person's struggle]..."*

## After Writing

Save to `.satori/artifacts/letters/YYYY-MM-DD.md`:

```markdown
# Pattern Letter — YYYY-MM-DD
> Context: [what prompted this letter — 1 sentence]
> Pattern addressed: [the specific struggle]

[Full letter text]
```

Note in session journal that a Pattern Letter was created and what pattern it addressed.
